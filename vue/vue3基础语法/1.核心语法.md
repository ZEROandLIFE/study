# Vue3 基础模板语法示例

## 核心指令与语法

### 1. 基础模板结构
```html
<template>
  <!-- 重点掌握这些就够了 -->
  <div>
    <!-- 1. 文本显示 -->
    <h1>{{ message }}</h1>
    
    <!-- 2. 条件渲染 -->
    <div v-if="isVisible">显示内容</div>
    
    <!-- 3. 列表渲染 -->
    <li v-for="item in list" :key="item.id">
      {{ item.name }}
    </li>
    
    <!-- 4. 事件处理 -->
    <button @click="handleClick">点击</button>
    
    <!-- 5. 属性绑定 -->
    <input :value="inputValue" :placeholder="placeholder">
  </div>
</template>
```

### 2. 基础数据展示
```html
<template>
  <div>
    <!-- 文本插值 -->
    <p>{{ message }}</p>
    
    <!-- 表达式 -->
    <p>{{ count + 1 }}</p>
    <p>{{ isActive ? '激活' : '未激活' }}</p>
    
    <!-- 原始 HTML -->
    <div v-html="rawHtml"></div>
  </div>
</template>
```

## 配套脚本 (Script Setup)
```html
<script setup lang="ts">
import { ref } from 'vue'

// 响应式数据
const message = ref('Hello Vue 3')
const count = ref(0)
const isActive = ref(true)
const rawHtml = ref('<span style="color: red">红色文字</span>')

// 示例方法（需在模板中使用的需要补充）
const handleClick = () => {
  console.log('按钮被点击')
}
</script>
```

## 关键语法说明

1. **文本插值**  
   `{{ message }}` - 使用双大括号显示响应式数据

2. **指令系统**  
   - `v-if` - 条件渲染  
   - `v-for` - 列表渲染（务必添加 `:key`）  
   - `v-html` - 渲染原始 HTML（注意 XSS 风险）

3. **动态绑定**  
   - `:value` - 属性绑定简写  
   - `@click` - 事件监听简写

4. **表达式支持**  
   可以在插值中使用简单表达式：  
   ```html
   {{ count + 1 }}
   {{ isActive ? '激活' : '未激活' }}
   ```
