# Vue.js 学习记录

## 学习内容概览

系统学习了 Vue 3 的完整知识体系，从基础语法到高级特性全面掌握：

### Vue 基础语法
- **核心语法**：Vue 实例和模板基础
- **数据绑定**：属性绑定、双向数据绑定
- **条件渲染**：v-if、v-show 的使用场景
- **列表渲染**：v-for 循环和 key 的作用
- **事件处理**：方法定义和事件修饰符
- **表单输入**：各种表单元素的数据绑定

### 响应式系统
- **setup 语法**：Composition API 的入口
- **ref 响应式**：基本类型的响应式数据创建
- **reactive 响应式**：对象类型的响应式处理
- **ref 与 reactive 区别**：使用场景和选择依据
- **toRefs**：解构响应式对象的工具函数

### 计算与监视
- **computed 计算属性**：派生状态管理
- **watch 监视**：ref 的两种监视方式
- **reactive 监视**：对象数据的深度监视
- **对象内部数据监视**：精确控制监视范围
- **数组监视**：数组变化的响应式处理
- **watchEffect**：自动依赖追踪的副作用

### 组件开发
- **组件通信**：9种不同的组件通信方式
- **Props 传递**：父子组件数据传递
- **ref 传递**：组件实例的引用
- **生命周期**：Vue 3 的生命周期钩子函数

### 路由管理
- **RouterView 和 Link**：路由组件和导航
- **路由基础**：路由配置和基本使用
- **Query 传参**：URL 查询参数操作
- **Params 传参**：动态路由参数传递
- **路由 Props**：路由配置中的属性传递
- **编程式导航**：通过代码控制路由跳转

### 状态管理
- **Pinia 基础**：状态存储和数据调用
- **Pinia 高级**：状态监听、getter 和多种写法
- **数据持久化**：状态管理的完整流程

### 高级特性
- **浅层响应式**：shallowRef 与 shallowReactive
- **只读处理**：readonly 与 shallowReadonly
- **原始数据**：toRaw 与 markRaw 的使用
- **自定义 Ref**：customRef 的定制化响应式
- **Teleport**：组件模板传送
- **Suspense**：异步组件加载状态

### 网络请求
- **Axios 使用**：HTTP 请求库的集成
- **接口封装**：API 请求的统一管理
- **Hooks 封装**：自定义组合式函数的创建

## 技术亮点
- 全面掌握 Vue 3 Composition API
- 深入理解响应式原理和实现机制
- 熟练运用组件通信和状态管理方案
- 掌握路由配置和导航控制
- 具备封装可复用逻辑的能力
- 了解 Vue 3 各种高级特性和优化技巧
