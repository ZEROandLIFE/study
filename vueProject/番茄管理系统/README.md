# 🍅番茄管理系统 - 前端项目

一个功能完整的企业级后台管理系统，基于现代前端技术栈开发，提供用户管理、商品管理、权限控制等核心功能。

git链接：https://github.com/ZEROandLIFE/Backend-Management-System

##  项目特色

- **现代化技术栈**：采用 Vue 3 + TypeScript + Element Plus + SCSS 构建
- **完整功能模块**：涵盖用户、商品、品牌、权限等管理功能
- **响应式设计**：支持多端适配，提供良好的用户体验
- **主题切换**：支持深色/浅色主题动态切换
- **数据可视化**：集成 ECharts，提供丰富的数据展示
- **Mock 数据**：内置完整的模拟数据，开箱即用

## 项目结构
采用模块化设计，按功能模块组织代码结构，确保代码的可维护性和可扩展性。
```├
mock/                  # Mock 数据
├── brand.ts          # 品牌 Mock 数据
├── product.ts        # 商品 Mock 数据
└── user.ts           # 用户 Mock 数据

src/
├── api/                    # API 接口定义
│   ├── product/           # 商品相关 API
│   │   ├── index.ts       # 商品 API 方法
│   │   └── type.ts        # 商品数据类型定义
│   ├── user/              # 用户相关 API
│   │   ├── index.ts       # 用户 API 方法
│   │   └── type.ts        # 用户数据类型定义
│   └── ...
├── assets/                # 静态资源
├── layout/                # 布局组件
├── router/                # 路由配置
├── store/                 # 状态管理
│   └── modules/
│       ├── user.ts       # 用户状态
│       └── product.ts    # 商品状态
├── styles/                # 样式文件
│   ├── index.scss        # 主样式文件
│   ├── reset.scss        # 重置样式
│   └── vars.scss         # 样式变量
└── views/                 # 页面组件
    ├── 404/              # 404 页面
    ├── acl/              # 权限管理
    ├── home/             # 首页
    ├── login/            # 登录页
    ├── product/          # 商品管理
    └── screen/           # 数据大屏
 ```

##  核心功能

###  用户认证系统
- 用户登录/登出
- Token 身份验证
- 路由权限守卫
- 自动登录状态管理

###  用户管理
- **普通用户管理**：查看用户列表、基本信息
- **管理员管理**：管理员账户的增删改查
- **权限分配**：角色权限管理
- **个人信息**：头像、密码等个人信息维护

###  商品管理
- **商品列表**：完整的商品信息展示
- **商品搜索**：支持按名称、品牌、标签多维度搜索
- **商品详情**：详细的商品信息页面
- **库存管理**：实时库存监控和预警
- **标签管理**：商品标签分类体系

### 品牌管理
- 品牌信息维护
- 品牌 Logo 管理
- 品牌与商品关联
- 品牌数据统计分析

###  数据大屏
- 实时数据监控
- 销售趋势分析
- 用户分布统计
- 渠道转化率分析
- 全国数据地图展示

## 技术架构

### 前端技术栈
- **框架**：Vue 3 + Composition API
- **语言**：TypeScript
- **UI 组件**：Element Plus
- **状态管理**：Pinia
- **路由管理**：Vue Router
- **构建工具**：Vite
- **数据可视化**：ECharts
- **样式预处理器**：Sass/SCSS

### 开发特性
- **类型安全**：完整的 TypeScript 类型定义
- **组件化开发**：高度可复用的组件设计
- **响应式状态**：基于 Pinia 的现代化状态管理
- **路由拦截**：完整的权限验证机制
- **Mock 数据**：开发环境数据模拟

## 🔧 开发指南

### 环境要求
- Node.js 16+
- npm 或 yarn
- 现代浏览器支持

### 启动项目
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📝 使用说明

5. **数据查看** → 在数据大屏查看统计分析
