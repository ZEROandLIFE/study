> 大二完成的综合性电商平台项目，集商品交易、社交互动于一体。大二简单完成的番茄商城，实现本地mysql存储和阿里云上传图片的功能（没续费，已过期）同时这是学习前端的一个原因————看起来比较有趣。

##  项目特色

-  **完整电商功能** - 商品浏览、购物车、订单支付一体化
-  **社交书友圈** - 集成社区互动，支持发帖、评论、点赞
-  **角色权限管理** - 顾客与管理员双角色系统
-  **响应式设计** - 适配多端设备的现代化UI
-  **云端图床** - 集成阿里云OSS图片上传（原功能，现无续费，已无法上传图片）

### UI组件库
- **Element Plus** - 基于Vue 3的组件库

## 核心功能模块

### 1. 用户系统 
- **用户认证** - JWT token登录验证
- **角色管理** - 顾客(USER)和管理员(ADMIN)双角色
- **个人信息** - 资料修改、头像上传、密码更改
- **会话管理** - sessionStorage存储用户状态

### 2. 商品系统 
- **商品展示** - 列表展示、详情查看、商品搜索
- **商品管理** - 管理员可创建、修改、冻结商品
- **商品评分** - 用户评分系统(1-5分)
- **库存管理** - 实时库存显示和限制

### 3. 购物车系统 
- **购物车操作** - 添加商品、数量调整、批量删除
- **选中管理** - 全选/反选、独立选择
- **实时计算** - 自动计算总价和选中数量
- **库存校验** - 购买时校验库存状态

### 4. 订单系统 
- **订单创建** - 支持购物车批量下单和直接购买
- **支付集成** - 支付宝沙箱支付
- **订单状态** - 状态跟踪和管理
- **历史记录** - 购买记录持久化存储

### 5. 书友圈系统 
- **圈子管理** - 创建圈子、加入/退出、权限管理
- **内容互动** - 发帖、评论、点赞功能
- **社交关系** - 圈主、管理员、普通成员角色
- **内容收藏** - 帖子点赞和收藏列表
 
### 6. 后台管理 
- **商品管理** - 商品CRUD操作、库存管理
- **广告管理** - 广告位配置和内容管理
- **用户管理** - 用户信息查看和管理
- **数据统计** - 基础数据展示

##  开发特性

### API设计
- **模块化组织** - 按功能模块拆分API文件
- **统一前缀** - 通过`_prefix.ts`统一管理接口前缀
- **类型安全** - 完整的TypeScript类型定义
- **错误处理** - 统一的请求拦截和错误处理


### 路由守卫
```typescript
// 路由权限控制
router.beforeEach((to, from, next) => {
  const token = sessionStorage.getItem('token')
  const role = sessionStorage.getItem('role')
  
  // 登录状态验证和权限控制
  if (to.meta.permission && !to.meta.permission.includes(role)) {
    next('/404')
  }
})
```
