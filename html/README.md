# HTML & CSS 学习记录

## 学习内容概览

通过系统学习，掌握了从基础到进阶的前端开发技能：

### 基础篇
- **HTML基础结构**：文字、图片、音频的嵌入，列表与表格的创建
- **表单元素**：输入框、菜单、文本域、按钮等交互控件
- **CSS基础选择器**：类选择器、ID选择器、标签选择器等
- **样式设置**：字体样式、颜色配置、文本装饰

### 进阶篇
- **复杂选择器**：伪类选择器、伪元素选择器的应用
- **CSS三大特性**：层叠性、继承性、优先级
- **显示模式**：块元素、行内元素、行内块元素的特性与转换
- **盒子模型**：margin、padding、border的详细使用
- **背景控制**：多背景、渐变、定位等高级背景效果

### 布局技术
- **传统布局**：浮动布局、定位布局的实战应用
- **现代布局**：Flex弹性盒模型的完整掌握
- **定位系统**：相对定位、绝对定位、固定定位的运用场景

### 动画与特效
- **过渡动画**：平滑的状态过渡效果
- **2D变换**：平移、旋转、缩放、倾斜等平面动态效果
- **3D变换**：立体空间中的动画效果实现
- **关键帧动画**：复杂动画序列的创建与控制

- 熟练运用CSS3新特性创建丰富的视觉效果
- 能够实现复杂的布局结构和动画效果
