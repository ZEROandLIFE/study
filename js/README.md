# JavaScript 学习记录

## 学习内容概览

通过实践项目驱动的方式，系统掌握了 JavaScript 核心知识和 DOM 操作技能：

### 基础语法与功能
- **输入输出**：掌握基本的用户交互方式
- **函数进阶**：箭头函数的使用和 this 指向理解
- **数组操作**：map、join 等数组方法的应用
- **解构赋值**：数组解构的便捷用法

### DOM 操作核心
- **元素获取**：多种 DOM 元素匹配方法
- **样式修改**：动态修改元素样式和类名
- **事件处理**：点击、输入、焦点等事件绑定
- **事件流机制**：事件捕获和冒泡的深入理解

### 定时器与动画
- **定时器应用**：setTimeout 和 setInterval 的使用
- **轮播图实现**：经典的动态效果开发
- **交互控制**：全选、单选等表单控制逻辑

### BOM 浏览器对象模型
- **窗口控制**：浏览器窗口相关操作
- **location 对象**：URL 管理和页面跳转
- **本地存储**：localStorage 的数据持久化

### 实战项目案例
- **回到顶部**：页面滚动交互功能
- **日期处理**：时间相关的操作和显示
- **DOM 节点操作**：元素的增删改查
- **广告弹窗**：结合延时器的用户交互设计
- **新增案例**：综合性的数据管理应用
- **标签切换**：多标签页面的交互实现

## 技术亮点
- 熟练运用 DOM API 进行页面动态交互
- 掌握事件驱动编程思想
- 能够实现复杂的用户交互逻辑
- 具备数据持久化和本地存储能力
- 通过实战项目巩固了 JavaScript 核心概念
