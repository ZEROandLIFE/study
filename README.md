# 前端学习仓库

> 一名大学生系统学习前端开发的过程记录，各种详情，请见文件夹中README

## 📚 学习路线
HTML → JavaScript → AJAX → Vue.js → Vue项目实战->
(React 待学习)

## 📖 学习内容

### 1. HTML 学习
从基础到动画的完整学习路径：
- **HTML 基础**：标签、语义化、表单
- **CSS 基础**：选择器、盒模型、定位
- **CSS3 进阶**：Flexbox、Grid、动画、过渡
- **移动端**：暂未学习

### 2. JavaScript 学习
重点掌握 DOM 操作和核心概念：
- **基础语法**：变量、数据类型、运算符
- **函数与作用域**：闭包、this、箭头函数
- **DOM 操作**：元素选择、事件处理、动态修改
- **定时器**：setTimeout、setInterval
- **ES6+ 特性**：模块化、Promise、async/await
- **使用DOM实现**：标签切换，全选控制，轮播图，广告等基本使用

### 3. AJAX 学习
前后端数据交互：
- **axios** 库的使用
### 4. Vue.js 学习
现代前端框架深入理解：
- **核心概念**：响应式、组件化、生命周期
- **模板语法**：指令、插值、事件处理
- **组件通信**：Props、自定义事件
- **Composition API**：ref、reactive、watch
- **状态管理**：Pinia 的使用
- **Hooks** 自定义组合式函数
- **路由**：Vue Router 配置
- **工具链**：Vite、Vue DevTools
- **额外常见学习**：EL组件库使用，配置拦截器，api分块管理，自定义组件，组件化实际应用(详见vueproject)
  
### 5. Vue 实战项目
综合应用所学知识：
- 项目需求分析与设计
- 组件化开发实践
- 状态管理方案
- 路由配置与权限控制
- 第三方库集成
- 性能优化技巧(未在项目中配置使用)

## 🛠️ 技术栈

- **前端三件套**：HTML5 + CSS3 + JavaScript(ES6+)/TS
- **构建工具**：Vite、Webpack
- **前端框架**：Vue 3 + Composition API
- **状态管理**：Pinia
- **路由管理**：Vue Router
- **HTTP 客户端**：axios
- **CSS 方案**：SCSS、Tailwind CSS
- **版本控制**：Git + GitHub
- **后端**：mock模拟后端进行api调用
